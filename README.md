# 🚀 TP N° 4 - JavaScript ES6+ - DOM, Eventos y APIs

## Descripcion
Este Proyecto es una mini aplicacion web desarrollada como trabajo practico para la materia Practicas Profecionales II.
El objetivo es aplicar JavaScript moderno (ES6+), manipulación del DOM, eventos y consumo de APIs públicas mediante fetch y async/await.

---

## 📌 Funcionalidades 

### 1. Ejercicios JavaScript
Se implementaron funciones usando:

- reduce() → cálculo de promedio
- filter() → filtrado de aprobados
- map() → formateo de datos
- find() → búsqueda de elementos

## 2. Sistema de Productos 
Página interactiva que permite:

- Visualizar productos en tarjetas
- Filtrar por categoría
- Filtrar por precio máximo
- Mostrar solo productos en stock
- Buscar productos por nombre en tiempo real

## 3. To-Do App
Lista de tareas funcional con:

- Agregar tareas
- Marcar como completadas
- Eliminar tareas
- Contador dinámico de pendientes
- Validación para evitar tareas vacías

## 4. Consumo de API (PokéAPI)
Se utilizó la API pública de Pokémon:

- Fetch con async/await
- Manejo de errores (try/catch)
- Validación de response.ok
- Estado de carga ("Cargando...")
- Renderizado dinámico de datos

## 5. Buscador dinámico
* Búsqueda en tiempo real
* Mínimo 3 caracteres
* Mensajes de estado:
 * "Buscando..."
  * "No se encontraron resultados"

## 6. Estilos CSS
- Uso de variables CSS (:root)
- Diseño responsive con Grid
- Tarjetas con sombras y hover
- Estilos consistentes en todas las páginas

---

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript ES6+
- Fetch API
- PokéAPI (https://pokeapi.co/)
